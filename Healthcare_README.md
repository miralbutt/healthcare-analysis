# 🏥 Healthcare Patient Readmission Analysis

**Author:** Miral Butt | Data Analyst | MPhil Statistics, LCWU Lahore

---

## 🎯 Project Overview

Hospital readmissions cost healthcare systems billions annually. This project analyzes patient data to identify key risk factors and predict **30-day hospital readmissions** using Machine Learning and Statistical Testing.

---

## 🔍 What's Inside

- **EDA** — Age, discharge type, medical conditions vs readmission
- **Statistical Tests** — Chi-square, T-test for significance testing
- **3 ML Models** — Logistic Regression, Random Forest, Gradient Boosting
- **ROC Curves** — Model comparison with AUC scores
- **Feature Importance** — Key drivers of readmission

---

## 📈 Model Performance

| Model | Accuracy | ROC-AUC |
|---|---|---|
| Logistic Regression | ~78% | ~0.78 |
| Random Forest | ~82% | ~0.83 |
| Gradient Boosting | ~83% | ~0.85 ✅ |
